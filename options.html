Web Automation

Paste text directly into ChatGPT or Gemini's web interface — no API key needed.

ℹ️ How It Works

When the API Type is set to ChatGPT Web or Gemini Web, the extension opens (or reuses) a tab for the respective service and pastes each chunk into the input box for you to send manually.

Delays: ChatGPT waits 2 s after load · Gemini waits 3 s after load.
Requirements: You must be logged in to the service in your browser.
